# Continuity

Repo-local continuity per `gov-09-agent-continuity-bootstrap.mdc`.

## Layers

| Layer | Path | Purpose |
|-------|------|---------|
| Session / thread | `session/` | Active chat/thread scratchpad; ephemeral |
| Recent / daily | `recent/` | Short-horizon decisions and open loops |
| Project | `project/CONTINUITY.md` | Durable project context |
| Operator / global | *(optional, out of repo)* | Cross-project norms — only when applicable |

## Operating docs

- `CHECKPOINT_GUIDANCE.md` — when to checkpoint
- `SESSION_DIARY_GUIDANCE.md` — recurring thread diaries
- `PROMOTION_GUIDANCE.md` — moving facts up layers

## Principle

Prefer concise factual artifacts over transcript dumps. Checkpoint during work, not only at the end.
